Steps to Run on GPU Machine

conda env update -f environment.yml
conda activate xlpsr
bash setup_project.sh
bash run_pipeline.sh
python3 code/apply_fine_tunes.py
Open code/ultraplate_pipeline.ipynb in your Jupyter environment and run all cells.
